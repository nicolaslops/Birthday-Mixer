# Birthday Mixer — Unified Professional Prototype

Includes:
- OpenStreetMap (Leaflet) maps
- Login/signup (client-side SHA-256 hashing prototype)
- Admin mode (admin@mix.com / admin123) with Chart.js graphs
- Premium simulated payments
- RSVP, contact requests, privacy-preserving counts

Run locally:
1. Extract ZIP and open folder in VS Code.
2. Use Live Server to serve files (recommended).
3. Visit `signup.html` to create user (>=16 yrs) or `login.html` to sign in.
4. Admin: login with admin@mix.com / admin123 to see stats.
